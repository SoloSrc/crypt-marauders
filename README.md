# Crypt Marauders

A simple game about exploring crypts and fighting off Undead.

## How to build

The project can be opened on the Godot 4 editor. Assets are not included, so
download them from the links provided below and extract them to the `res/`
folder.

 * [Kay Kit Adventurers](https://kaylousberg.itch.io/kaykit-adventurers);
 * [Kay Kit Skeletons](https://kaylousberg.itch.io/kaykit-skeletons);
 * [Kay Kit Dungeons Remastered](https://kaylousberg.itch.io/kaykit-dungeon-remastered);
 * [Kay Kit Character Animations](https://kaylousberg.itch.io/kaykit-character-animations).
